# Bug Report - Add to Cart Not Working for Multiple Products

## Bug ID
BUG-001

## Related Test Case
TC_CF_001

## Description
The "Add to Cart" functionality fails for several inventory products.

## Steps to Reproduce
1. Login with valid credentials (username : error_user)
2. Navigate to products page
3. Click "Add to Cart" for multiple products

## Expected Result
Selected products should be added to the shopping cart successfully.

## Actual Result
Products are not added to the cart and cart badge count does not update.

## Severity
Major 

## Priority
P1
